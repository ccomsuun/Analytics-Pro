# 📊 Analytics Pro: Business Dashboard
**엑셀만 올리면 AI가 보고서까지 써주는 데이터 분석 도구**

## 🇰🇷 프로젝트 소개
이 툴은 한국의 일반적인 업무 관리 엑셀 파일을 업로드하면, 자동으로 핵심 지표(KPI)를 추출하고 시각화 차트를 생성합니다.

## ✨ 주요 기능
- **자동 헤더 감지**: 담당자, 상태, 날짜를 스스로 파악하여 분석합니다.
- **AI 리포트 (Simulated)**: 수집된 데이터를 바탕으로 비즈니스 인사이트를 생성합니다.
- **보고서 저장**: 분석된 대시보드를 PDF로 즉시 저장할 수 있습니다.
